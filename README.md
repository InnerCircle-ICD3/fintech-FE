## 지원 환경

- Node.js 22.9.0 이상
- 모든 최신 브라우저 (Chrome, Firefox, Safari, Edge)
- TypeScript 5.8.0 이상

## 개발 환경 설정

- pnpm workspace를 사용한 모노레포로 구성되어 있으니 pnpm부터 설치해주세요

```bash
# 의존성 설치
pnpm install

# SDK
pnpm run sdk

# Demo
pnpm run demo
```
